Shortcodes:
= [pt-stimm] =
Zeigt die Tabelle mit dem Abstimmverhalten an.
Parameter:
- id: ID des Gremiums
- ext: ID des externen Gremiums
- hidefrom: Automatisches Einklappen älterer Sitzungen, (Nummer der Sitzung, die als erstes eingeklappt werden soll)
= [pt-stimm-legende] =
Zeigt eine Legende an.
